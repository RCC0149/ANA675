# ANA 675 – Neural Networks & Deep Learning

**Author:** Randall C. Crawford

---

## Course Overview

This repository contains coursework completed for **ANA 675: Neural Networks & Deep Learning**, a graduate-level course focused on the **theory, architecture, and practical implementation of neural network and deep learning models** using Python.

The course emphasized understanding **how neural networks learn**, how architectural choices affect performance, and how deep learning systems are **built, trained, evaluated, and tuned** for real-world applications. Coursework progressed from classical machine learning concepts into modern deep learning frameworks and applications.

---

## Key Topics Covered

### Machine Learning Foundations

* Supervised vs. unsupervised learning
* Classification vs. regression problems
* End-to-end learning system design workflow

### Neural Network Architecture

* Artificial Neural Networks (ANNs)
* Activation functions and loss functions
* Backpropagation and gradient-based optimization
* Overfitting, regularization, and dropout

### Deep Learning Models

* Fully connected deep neural networks
* **Convolutional Neural Networks (CNNs)** for image classification
* **Recurrent Neural Networks (RNNs)** for sequence data
* **Long Short-Term Memory (LSTM)** networks for sentiment analysis

### Model Training & Evaluation

* Data preprocessing and normalization
* Train/validation/test splitting
* Performance metrics and confusion matrices
* Hyperparameter tuning and optimization strategies

---

## Hands-On Implementation

Assignments and projects required implementing models in **Python using TensorFlow and Keras**, including:

* Building and training ANNs on structured and image datasets
* CNN-based classification using datasets such as **CIFAR-10**
* RNN/LSTM models for **text and sentiment analysis**
* Visualization of training dynamics and evaluation results

The work emphasized **practical experimentation**, interpretability of results, and understanding trade-offs between model complexity and performance.

---

## Learning Emphasis

The course focused on:

* Translating real problems into neural network architectures
* Understanding **why deep learning models succeed or fail**
* Diagnosing training behavior and convergence issues
* Applying deep learning responsibly to real-world data

---

## Outcome

This repository demonstrates applied competency in **neural networks, deep learning architectures, and model evaluation**, providing a strong foundation for advanced work in **machine learning, artificial intelligence, and applied data science**.
